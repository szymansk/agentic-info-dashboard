🧩 Extensions & Plugins MCP + Claude Code · Mai 2026

Kuratierte MCP-Server (cross-tool) und Claude-Code-Plugins für produktiveres Arbeiten — Office (pptx/docx/Calendar/Mail), Code/Dev (Git/Browser/Testing/DB) und Productivity (Notion/Linear/Jira). Quellen-Marker (offiziell, Partner, Community) je Eintrag.

Kontext. MCP wurde im Dezember 2025 von Anthropic an die Linux Foundation Agentic AI Foundation übergeben — joint governance mit OpenAI, Google, Microsoft, AWS, Cloudflare, Bloomberg. Ökosystem hat im Mai 2026 die 14.000-Server-Marke überschritten. Der offizielle Marketplace für Claude Code zählt 101 Plugins (33 Anthropic, 68 Partner). Diese Sammlung kuratiert die produktivsten Einträge — keine vollständige Liste.
